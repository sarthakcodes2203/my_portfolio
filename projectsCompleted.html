To-Do Application
This React-based To-Do application provides full CRUD (Create, Read, Update, Delete) functionality, allowing users to efficiently manage their tasks. The app utilizes **local storage** to persist data, ensuring tasks remain saved even after a page refresh. Users can add, edit, and delete tasks effortlessly, with an intuitive interface designed for a smooth experience. Additionally, the app includes a **toggle feature** to show or hide completed to-dos, helping users focus on pending tasks. This project highlights strong frontend development skills and emphasizes user-friendly design for effective task management.

Medical Application
The Flutter based Medical Application is a cross-platform healthcare management system designed to streamline doctor appointments, medication purchases, and administrative tasks. Built with Flutter and Firebase, it offers secure user authentication, doctor categorization, and an intuitive appointment booking system with real-time availability tracking. Patients can search for doctors, book or cancel appointments, and order medications seamlessly. The app also features inventory management for medications, notifications for reminders, and report generation for administrators. Future enhancements like telemedicine integration, AI-based recommendations, and globalized healthcare access aim to further improve efficiency and accessibility in modern healthcare.
